# Controllo Diff

Leggi questo diff e valuta se risponde solo al task richiesto.

Controlla:

- cosa e' cambiato;
- se ci sono modifiche fuori scope;
- se il diff e' piccolo e leggibile;
- se manca una verifica importante;
- quali righe meritano review.
- se la strategia prompt era proporzionata: zero-shot o few-shot;
- se le evidenze riportate sono sufficienti per autorizzare o rifiutare il diff.

Rispondi in modo breve e operativo.
