| File | Uso |
| --- | --- |

## Regola

Questi file preparano un workflow kit riusabile. In L04 verranno copiati/adattati nella repo di codice.

Durante L03 non modifichi la repo di codice; nel lab subito dopo si'.

Il `prompt-kit/` raccoglie procedure riusabili per orientamento, piano, modifica controllata, diff e handoff.

Il kit deve restare leggero: esempi few-shot solo quando chiariscono formato o criterio, evidenze brevi al posto di chain-of-thought estesa.
